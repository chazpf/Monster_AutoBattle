Monster AutoBattle

Welcome to Monster AutoBattle!

This is a simple autobattle game using the Open5e api to access the Dungeon & Dragons Fifth Edition SRD monster information made public by Wizards of the Coast.

This project was created for educational purposes, as I am learning web development. I do not own nor claim to own any of the images used in this project.

Please press “Start Game” and enjoy your stay.
